Задание 2 (сложность 3)

Реализовать ранее спроектированную классовую иерархию из Занятия 1 Задание 2. Все классы должны иметь конструкторы,
содержащие внутри вывод на имени класса. Написать тестовый класс, создающий объекты всех классов получившейся иерархии
и заолняющий нужные ссылки объектов друг на друга. Скомпилировать и проверить работоспособность программы.

Требования к выполнению:

1) Исходный .java файл и откомпилированный .class файл должны быть "вкомитаны" в GIT
2) Использование для выполнения любых IDE не допускается.
3) Для вывода сообщения использовать - System.out.println("ТЕКСТ_СООБЩЕНИЯ");
